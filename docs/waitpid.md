# waitpid

Функция waitpid приостанавливает выполнение текущего процесса до тех пор, пока
дочерний процесс, указанный в параметре pid, не завершит выполнение, или пока не
появится сигнал, который либо завершает текущий процесс либо требует вызвать
функцию-обработчик. Если указанный дочерний процесс к моменту вызова функции уже
завершился (так называемый "зомби"), то функция немедленно возвращается.
Системные ресурсы, связанные с дочерним процессом, освобождаются.

```
#include <sys/types.h>

pid_t waitpid(pid_t pid, int *status, int options);  
```
